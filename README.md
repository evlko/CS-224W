# CS-224W
Social and Information Network Analysis

## Lab 1: VK API
* Selecte 2 communities;
* Get members and posts from them via VK API;
* Analyze their hashtags & posts time;
* Clusterize the union of two communities;
* Plot the graph.

## Lab 2: Graph Analysis
* Edges & Verticies;
* Components;
* Density;
* Modularity;
* Centrality.
